# lelang
leang是一门使用Rust编写，基于LLVM(inkwell llvm safe binding library)实现的编程语言，起初作为课程实验项目，现在为个人长期维护项目。

### Target Features

+ 支持8至64位的整形类型和32/64位浮点
+ 基本的函数定义，调用，声明外部函数语法
+ 链接外部函数
+ 可解释或编译执行
+ 变量定义
+ 静态类型，编译期检查类型
+ C like语法
